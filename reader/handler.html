Subscribe

This page is the general 'subscribe' page for the reader. It is called by the subtome page with the resource query string param that includes the URL of the page on which the user clicked the "Subscribe" button.

This app is then in charge of extracting whatever data it want to actually perform the subscription. It can be an RSS feed, or other custom elements, like for example a twitter handle, a facebook handle... etc or anything to which this app can susbcribe the user!
